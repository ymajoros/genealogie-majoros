## Note historique sur Békásmegyer

**Békásmegyer** = quartier nord de Budapest où se trouvait un **camp de concentration / point de regroupement** pour la déportation des juifs d'Újpest pendant l'été 1944. Le décès « à Békásmegyer 1944 » d'Elza Elek est cohérent avec l'enchaînement : rafles Újpest mai-juin 1944 → ghetto/Sammellager Békásmegyer (juin-juillet 1944) → train vers Auschwitz fin juillet 1944.

Ce contexte renforce la probabilité que Bokor Panni ait suivi le même parcours : Újpest → Békásmegyer → Auschwitz.
