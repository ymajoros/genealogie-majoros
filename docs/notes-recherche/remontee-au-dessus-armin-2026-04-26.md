## 4. Hypothèses ouvertes pour le cousinage

### 4.1. Cousinage hypothétique Ármin ↔ szinóbányai

| Configuration | Plausibilité | Conséquence |
|---|---|---|
| **A. Ármin = neveu d'Ignácz** (père d'Ármin = frère d'Ignácz) | Modérée | Ármin = cousin germain de Jakab/Samu/Rezső/József ; Zsigmond = 2ᵉ cousin de Tivadar/Leó/Henrik/Felix |
| **B. Ármin = petit-neveu** (grand-père d'Ármin = frère d'Ignácz) | Modérée | Zsigmond = 3ᵉ cousin de Tivadar/Leó |
| **C. Ármin = neveu par alliance / pas de lien sang** | Plausible | Pas de cousinage effectif |
| **D. Patrony homonyme sans parenté** | Plausible aussi | Aucune relation |

**Pourquoi Ármin n'est PAS un fils d'Ignácz patriarche** :
- 1881 : Ignácz † liste 6 enfants vivants (Jakab, Rezső, Samu, József, Katalin, Cecília). Pas d'Ármin.
- Si Ármin était un 7ᵉ enfant prédécédé, il serait à minima nommé comme grand-père dans l'avis (par la mention « néhai testvére »). Absent.

**Pourquoi Ármin n'est PAS un petit-fils d'Ignácz** :
- 1881 : 18 petits-enfants vivants nommés (Tivadar, Hugó, Róza, Leo, Mór, Oszkár, Ilona, Vilma, Margit, Jenő, Henrik, Aranka, Félix, Adél, Ernő, Emil + Adám, Jenny, Klára, Károly, Miksa). **Aucun « Ármin »**.

→ Si lien existe, il passe par une **branche collatérale** (frère ou cousin d'Ignácz patriarche), donc l'ancêtre commun se situe à la **génération de la fin du XVIIIᵉ siècle** (les parents d'Ignácz, b ~1750-1770).

### 4.2. Géographie

- Ignácz patriarche : Pest **Nagy Korona-utca 24** (V. kerület, près de l'Opéra) — donc Pest centre-bourgeois ; origine antérieure plausible de Hongrie ou Moravie occidentale (familles juives ashkenazes germanophones).
- Notre Ármin : présumé **Nyitra/Komárom** (NW Hongrie historique, aujourd'hui Slovaquie ouest)
- Distance Pest ↔ Nitra : 165 km — pas excessive pour des cousins éloignés.

### 4.3. Onomastique

Les szinóbányai utilisent les prénoms **Jakab, Rezső, Samu, József** + leurs descendants : Jakab → Tivadar, Hugó, Leó, Mór, Oszkár ; Rezső → Jenő, Henrik ; József → Felix, Ernő, Emil, Paula, Adél ; Samu → Henrik, Aranka.

Notre Ármin nomme ses fils : **Zsigmond, László, Sámuel** + filles **Júlia, Róza**. **Zsigmond et László** ne figurent pas dans la lignée szinóbányai. **Sámuel** est commun (= Samu chez les anoblis). **Júlia** et **Róza** ne sont pas dans les szinóbányai.

→ Onomastique distincte, pas de transmission flagrante de prénoms.

## 6. Conclusion provisoire

**À ce jour (26 avril 2026), aucun lien de parenté entre notre Krámer Ármin et la famille szinóbányai Kramer n'est documenté par source primaire.** Les indices indirects (proximité géographique post-1900 à Pest district V/VI, professions bourgeoises juives, communauté ashkenaze NW-Hongrie) suggèrent une parenté collatérale plausible mais non démontrée. La fiabilité du blason armorié de 1906 (lions d'argent, roue dentée, étoiles d'or) reste **réservée à la branche szinóbányai uniquement**.

Pour conclure, il faudrait soit :
- Identifier l'**acte de naissance de Krámer Ármin** (~1840 Nyitra/Komárom) et lire les noms de ses parents
- Identifier l'**ascendance d'Ignácz Kramer (1789-1881)** via Geni Pro / NovaNobilitas / Wien
- Et chercher une intersection à la **génération ~1750-1780**

Sans ces deux éléments, la question du cousinage reste ouverte.
