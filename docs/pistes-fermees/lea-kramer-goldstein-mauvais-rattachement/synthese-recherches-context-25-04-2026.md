## H. Recettes techniques validées

- **Hungaricana ADT search-info** (compte exact) : `library.hungaricana.hu/adt-search-info/?query=...`
- **ADT result list** (paginé) : `adtplus.arcanum.hu/en/search/results/?query=...&page=N` — extraire `/view/DOC/?pg=N` par regex
- **OSZK gyászjelentések strict title** : `dspace.oszk.hu/discover?filtertype_0=title&filter_relational_operator_0=contains&filter_0=NAME&query=`
- **Kozma cemetery** : nonce dans `database.budapestjewishcemetery.com/hu/adatbazis-kereses/` puis `wp-admin/admin-ajax.php?action=db_call&_wpnonce=NONCE&oper=search&searchstr=`
